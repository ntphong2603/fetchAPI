Xây dựng một giao diện có thể xem và quản lý được Post + Comment

- Đã làm được:
    + GET and ADD POST (tạo 1 form để thêm thông tin POST)
    + GET COMMENT
- Chưa làm được:
    + ADD Comment
    + Chưa hiển thị được phần giao diện ADD COMMENT để 
    thêm thông tin vào COMMENT.


Giao diện mong muốn khi nhấn vào nút GET API POST DATA hoặc GET API COMMENT DATA thì sẽ
hiện ra 1 form để Thêm thông tin của POST hoặc COMMENT.
